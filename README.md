# TravelPlanner (Golobe) 🌍

> Full-stack system for trip planning and booking / System rezerwacji i planowania podróży / Система бронирования и планирования путешествий.

---

## 🇵🇱 Język Polski

### O projekcie

**TravelPlanner** to kompleksowe rozwiązanie webowe oparte na makiecie "Golobe". Platforma umożliwia użytkownikom pełną organizację podróży: od wyszukiwania lotów i hoteli po zarządzanie budżetem i harmonogramem wyjazdu.

### Stos technologiczny

- **Frontend:** React, TypeScript, Vite.
- **Stylizacja:** SCSS (Sass) z wykorzystaniem metodologii CSS Modules.
- **Backend:** Node.js (Express/Fastify).
- **Baza danych:** PostgreSQL.
- **Architektura:** Feature-Sliced Design (FSD).

### Główne funkcjonalności

- Wyszukiwarka lotów i noclegów z zaawansowanym filtrowaniem.
- Interaktywny kalendarz i plan podróży.
- System kont użytkowników i autoryzacja.
- Generowanie i przechowywanie biletów.

## 🇬🇧 English

### About the Project

**TravelPlanner** is a comprehensive web solution based on the "Golobe" design. The platform enables users to fully organize their trips: from searching for flights and hotels to managing budgets and travel itineraries.

### Tech Stack

- **Frontend:** React, TypeScript, Vite.
- **Styling:** SCSS (Sass) using CSS Modules.
- **Backend:** Node.js (Express/Fastify).
- **Database:** PostgreSQL.
- **Architecture:** Feature-Sliced Design (FSD).

### Key Features

- Flight and accommodation search with advanced filtering.
- Interactive calendar and trip itinerary planner.
- User account system and authentication.
- Ticket generation and storage.

## 🇷🇺 Русский

### О проекте

**TravelPlanner** — это комплексное веб-решение, основанное на макете "Golobe". Платформа позволяет пользователям полностью организовать свои поездки: от поиска авиабилетов и отелей до управления бюджетом и графиком путешествия.
